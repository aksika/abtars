# §4 Garbage Collection

Scan ALL messages in the DB and clean up noise. This is the most important maintenance task.

For any message you mark or delete, always include its paired assistant response:
```sql
SELECT id FROM messages WHERE id > <user_msg_id> AND role='assistant' ORDER BY id LIMIT 1;
```

## Step 1: Purge expired garbage

Read `~/.agentbridge/memory/garbage.json` (create `{}` if missing). Format: `{"<message_id>": "<ISO timestamp>"}`.
Delete entries whose garbage timestamp is older than 7 days:
```bash
agentbridge-store --delete-ids <comma_separated_ids> --chat-id <chat_id>
```
Remove those entries from `garbage.json` and write the file back.

## Step 2: Immediate deletes (no grace period)

**Duplicates:**
```sql
SELECT a.id, b.id FROM messages a JOIN messages b
ON a.chat_id = b.chat_id AND a.content = b.content AND a.id < b.id
AND abs(a.timestamp - b.timestamp) < 300000;
```
Keep first (lowest id), delete rest + paired responses.

**Wrong chat** — user says "wrong chat", "rossz chat". Delete the message, the one before it, and both responses.

**Whisper/STT garbage** — garbled transcriptions that make no sense in any language. Delete immediately + paired responses.

## Step 3: Repeated probes

```sql
SELECT content, COUNT(*) as cnt, GROUP_CONCAT(id) as ids
FROM messages WHERE role='user'
GROUP BY content HAVING cnt >= 3 ORDER BY cnt DESC;
```
Keep first occurrence + response. Mark rest as garbage in `garbage.json`.

## Step 4: Noise marking (grace period)

Mark as garbage messages with zero informational content:
- Single-word greetings: "hi", "hallo", "hello", "hey"
- Pings: "prof", "professor", "vagy prof?", "are you there?"
- Filler: "ja", "igen", "aha", "I see", "jaja", "Na nézzük"
- Single characters: "a", "?"

Do NOT mark: action confirmations, instructions, questions with content, facts, jokes, topic starters.

## Step 5: Verify extractions

Scan messages not yet captured in `extracted_memories`. For each exchange with lasting value:
1. Check if facts are already in extracted_memories
2. If not, extract via `agentbridge-store --translated "..." --original "..." --memory-type <TYPE> --emotion-score <SCORE> --chat-id <CHAT_ID> --trust 2 --integrity 2 --credibility 2`
3. After confirming stored, garbage-mark the verbose originals

**Extract:** facts, decisions, preferences, events, lessons, tool configs, workflow discoveries.
**Skip:** greetings, debugging noise, purely instructional exchanges.

## Step 6: Emotion harvest (verbal only)

Scan for verbal emotional reactions (emoji reactions are already handled at runtime):
- Positive: "fasza!", "király!", "awesome!", "excellent!", "nice!"
- Negative: "a faszomat!", "baszd meg!", "goddamn it!", "fuck!"

For each:
1. Find the nearest relevant extracted_memory
2. `agentbridge-edit --memory-id <N> --emotion-score <score> --caller dreamy` (+1 to +3 or -1 to -3)
3. Add message + paired response to `garbage.json`

## Step 7: Flush old messages

Delete all messages older than 24 hours. By this point, all valuable content has been extracted, summarized, and captured in the retrospective.

```sql
sqlite3 ~/.agentbridge/memory/memory.db "DELETE FROM messages WHERE timestamp < $(date -d '24 hours ago' +%s%3N);"
```

Respond with counts: deleted, marked, extracted, emotions harvested, flushed.
