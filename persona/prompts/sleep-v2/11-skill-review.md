# Skill Review

Review recent conversations for reusable patterns worth capturing as auto-skills.

## Task

1. Look for repeated workflows, commands, or problem-solving patterns from recent conversations.
2. For new patterns: `agentbridge-skill create --name "<name>" --trigger "<when to use>" --steps "<what to do>"`
3. For existing skills that need updating: `agentbridge-skill edit --name "<name>" --steps "<updated steps>"`
4. Do not create skills for one-off tasks.

Respond with skills created or updated.
